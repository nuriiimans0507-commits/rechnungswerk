Häufige Fragen

Ist das Rad wirklich fair?

Ja. Der Gewinner wird per Zufallsgenerator bestimmt – jede Option hat exakt dieselbe Chance, egal wo sie im Rad steht. Die Drehanimation visualisiert nur das Ergebnis.

Werden meine Einträge gespeichert?

Nur lokal in deinem Browser, damit deine Liste beim nächsten Besuch noch da ist. Es wird nichts an Server übertragen.

Kann ich einen Gewinner aus dem Rad entfernen?

Ja – nach jedem Dreh erscheint „Gewinner entfernen“. So kannst du z. B. bei einer Verlosung mehrere Plätze nacheinander ziehen, ohne dass jemand doppelt gewinnt.

Funktioniert das Glücksrad auf dem Handy?

Ja, das Rad passt sich jeder Bildschirmgröße an und läuft in jedem modernen Browser – ohne App, ohne Installation.
